yarn命令

发表于 2021-10-21   |   字数统计: 286(字)   |   阅读时长: 1(分)

yarn add

-E 选项明确指定版本
—audit 检查在已安装的包中所有已经的安全问题并输出

yarn autoclean

autoclean 命令通过从依赖项中删除不必要的文件和文件夹来释放空间。它减少了项目节点模块文件夹中的文件数量，这在直接将包签入版本控制的环境中非常有用。

yarn bin

输出 yarn 安装可执行文件的目录

yarn cache

可以管理下载的包缓存

yarn create
